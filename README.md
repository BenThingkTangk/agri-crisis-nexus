# AGRI·CRISIS NEXUS

**Strategic Intelligence Platform — Chief Quantum Officer Eyes Only**

A comprehensive agricultural crisis intelligence web application consolidating three previous intelligence briefings into a unified defense-grade command center.

## Access

Password-gated. Contact the Chief Quantum Officer for credentials.

## Modules

1. **Global Crisis Command Map** — Interactive Leaflet map with 6 threat layers
2. **Tactical Threat Radar** — D3 radar chart with 8-axis country assessment
3. **Geopolitical Chess Board** — Animated 8x8 board with actor cards
4. **Live Intelligence Feed** — 100 hardcoded intel cards from 11 sources
5. **Data Visualization Suite** — 6 Chart.js visualizations
6. **Social Impact Pulse** — 6 real-time intervention sliders
7. **Global Status Board** — Traffic light bubbles for 40+ regions
8. **Water & Aquifer Intelligence** — Ogallala countdown, virtual water trade
9. **Biotech & Soil Science** — CRISPR pipeline, regenerative ag ROI
10. **Strategic Timeline** — 2020–2050 zoomable event scroll
11. **Strategic Ops Matrix** — Nirmata positioning + confidence scores

## Stack

- Single-file HTML/CSS/JS
- Leaflet.js, Chart.js, D3.js, GSAP
- Fontshare (Clash Display, Satoshi), Google Fonts (Space Mono, Rajdhani)
- Deployed on Vercel

## Development

```
open index.html
```

No build step. All data hardcoded.
